# AI Recruitment & Screening System

## Folder Layout
- **data/**: Dataset storage
- **ats_engine/**: Resume matching logic
- **screening_ai/**: Candidate evaluation models
- **logs/**: Activity logs
parsers/: Data processing scripts.
​interview_ai/: Video/Audio analysis services.

​scoring/: Final decision and scoring logic.

​utils/: Helper functions and common tools.

​tests/: Unit tests and sample test scripts.

## Setup
1. Activate virtual environment.
2. Install dependencies: `pip install -r requirements.txt`
