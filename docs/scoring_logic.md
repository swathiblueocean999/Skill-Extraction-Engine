# Aptitude Scoring Logic

## Objective

Generate aptitude evaluation scores using reasoning, situational handling, clarity, and problem-solving analysis.

---

# Scoring Parameters

| Parameter | Weight |
|---|---|
| Reasoning Score | 20% |
| Scenario Handling | 20% |
| Clarity Score | 20% |
| Structure Score | 15% |
| Problem Solving | 15% |
| Decision Making | 10% |

---

# Final Formula

Final Aptitude Score =
(
Reasoning * 0.20
+
Scenario * 0.20
+
Clarity * 0.20
+
Structure * 0.15
+
Problem Solving * 0.15
+
Decision * 0.10
)

---

# Final Status

| Score | Result |
|---|---|
| 75+ | Strong |
| 50–74 | Average |
| Below 50 | Weak |
