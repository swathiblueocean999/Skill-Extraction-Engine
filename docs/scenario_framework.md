# Scenario Evaluation Framework

## Objective

Evaluate candidate situational judgment and workplace decision-making ability.

---

# Scenario Categories

## Client Handling
Evaluates professionalism while handling clients.

## Team Collaboration
Measures teamwork and cooperation skills.

## Stress Management
Checks calmness under pressure.

## Conflict Resolution
Measures handling of workplace disagreements.

## Deadline Management
Checks prioritization and responsibility.

---

# Evaluation Areas

| Area | Purpose |
|---|---|
| Calmness | Emotional control |
| Professionalism | Workplace maturity |
| Communication | Clear interaction |
| Problem Solving | Practical solution ability |
| Responsibility | Ownership mindset |

---

# Score Classification

| Score | Status |
|---|---|
| 75+ | Strong |
| 50–74 | Average |
| Below 50 | Weak |

# Framework Goal

Build AI-based situational judgment evaluation for HR aptitude interviews.
