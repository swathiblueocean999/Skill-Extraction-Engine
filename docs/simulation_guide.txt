Zecpath AI – Simulation Guide

--------
This guide explains how to run the complete AI hiring simulation using the prepared demo dataset.

========================

-----------------
datasets/
outputs/
docs/
src/

Required Files:
---------------
• job_descriptions/*.json

SIMULATION FLOW
================

STEP 1 — Load Job Description
------------------------------
Example:
• Accountant JD
• Junior Accountant JD
• Senior Accountant JD

System Action:
• AI loads required role
• Extracts required skills
• Prepares ATS criteria

====================================

STEP 2 — Load Candidate Dataset
--------------------------------
Candidate Categories:
• Strong Candidates
• Average Candidates
• Weak Candidates

System Action:
• Load candidate profiles
• Validate candidate IDs
• Validate resume structure

====================================

STEP 3 — Run ATS Simulation
----------------------------
Process:
• Resume parsing
• Skill matching
• Experience matching
• Education matching

Generated Output:
• ATS Score
• ATS Recommendation

====================================

STEP 4 — Run Screening AI
--------------------------
Process:
• Analyze screening answers
• Evaluate communication
• Evaluate role understanding

Generated Output:
• Screening Score
• AI Evaluation

====================================

STEP 5 — Run HR Interview AI
-----------------------------
Process:
• Behavioral analysis
• Stability analysis
• Teamwork evaluation
• Confidence analysis

Generated Output:
• HR Score
• HR Recommendation

====================================

STEP 6 — Run Technical AI
--------------------------
Process:
• Accounting concept validation
• GST knowledge check
• Taxation knowledge
• ERP understanding

Generated Output:
• Technical Score
• Technical Evaluation

====================================

STEP 7 — Generate Final Decision
---------------------------------
Decision Logic:
• Score > 80 → Selected
• Score 55–79 → Hold
• Score < 55 → Rejected

Generated Output:
• Final Recommendation
• Candidate Ranking
• AI Hiring Decision
