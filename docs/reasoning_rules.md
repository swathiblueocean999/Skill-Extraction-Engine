# Reasoning Evaluation Rules

## Objective

Evaluate candidate logical thinking and reasoning ability during aptitude interviews.

---

# Evaluation Parameters

## 1. Logical Explanation
Checks whether the candidate explains reasoning clearly.

## 2. Decision Making
Evaluates ability to make practical decisions.

## 3. Step-by-Step Thinking
Checks whether the answer follows a structured process.

## 4. Solution Focus
Detects problem-solving orientation.

## 5. Responsibility Awareness
Checks accountability and professional behavior.

---

# Scoring Logic

| Score Range | Evaluation |
|---|---|
| 80–100 | Strong reasoning |
| 50–79 | Moderate reasoning |
| Below 50 | Weak reasoning |
