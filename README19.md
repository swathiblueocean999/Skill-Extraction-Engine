# ATS System Documentation (Day 19)

## Overview
This project is an AI-based ATS (Applicant Tracking System) that processes resumes, extracts relevant data, and scores candidates based on job requirements.

## Features
- Resume text extraction
- Data cleaning
- Skill and experience detection
- Scoring system
- Performance optimization

## Pipeline
1. Extract resume text
2. Clean text
3. Detect entities (skills, experience)
4. Score candidate
5. Output results

## Folder Structure
- optimizer/ → processing modules
- data/ → input files
- output.txt → results
