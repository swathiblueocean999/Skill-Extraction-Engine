Resume Processing & Ranking System


📌 Project Overview


This project processes multiple candidate profiles and converts unstructured resume data into a structured format. It then evaluates each profile based on experience, relevance, and career gaps, and assigns a score for ranking.


🎯 Objectives


* Extract and segment resume data


* Identify key sections like Experience, Education, and Qualifications


* Evaluate relevance of experience


* Detect career gaps


* Assign scores and rank candidates


⚙️ Features


✅ Text segmentation (Skills, Education, Qualifications, Experience)


✅ Experience calculation


✅ Relevance detection


✅ Gap detection


✅ Scoring system


✅ Ranking of profiles


📂 Input


* Raw extracted text file containing multiple candidate profiles


📤 Output


* segmented_profiles.txt → Structured profiles


* ranked_profiles.txt → Profiles with scores and ranking


* final_profiles.xlsx → Structured Experience Summary table


🧠 Processing Steps


* Split profiles from raw text


* Segment into sections


* Extract experience details


* Apply relevance logic


* Detect career gaps


* Assign score


* Rank profiles


📊 Structured Experience Summary


Each profile includes:


* Profile ID


* Total Experience


* Relevance Status


* Gap Detection


* Score


*  Conclusion


The system successfully processes all profiles, converts them into structured data, and ranks candidates based on defined criteria.
