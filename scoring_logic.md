# Scoring Logic

## Factors Considered
- Skills match
- Experience level
- Keywords

## Example Scoring
- Excel → +25
- Accounting → +20
- Experience ≥ 3 years → +20

## Threshold
- Score ≥ 30 → Good candidate
- Score < 30 → Weak candidate

## Improvements
- Dynamic threshold
- Weighted scoring
- Semantic matching (future)
